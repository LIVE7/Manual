# Part 1 — 기초

## 1. 이 지표는 무엇인가?

### 한 줄 요약
여러 기술지표(MACD, RSI, ATR, ADX 등)를 내부에서 계산해서
**"지금 사세요(▲)" 또는 "지금 파세요(▼)"를 화살표 하나로** 보여주는 도구입니다.

### 이 지표로 할 수 있는 것
- ✅ 매수(롱)/매도(숏) 타이밍 신호
- ✅ 이익실현(✕) 타이밍 신호
- ✅ 추세 방향 시각화 (캔들 색상, 오버레이 선)
- ✅ 자동 손절/이익실현 수준 계산 (레버리지 연동)
- ✅ 지지선/저항선 자동 표시
- ✅ 소리/앱 알림
- ✅ 현물 전용 모드 (숏 신호 숨김)

### 할 수 없는 것
- ❌ 100% 정확한 예측 (불가능합니다)
- ❌ 자동 매매 (신호만 보여줌, 주문은 직접)
- ❌ 뉴스/이벤트 반영

# Part 2 — 진입과 청산

## 4. ★ 정확한 진입 시점

### 핵심 규칙 (꼭 외우세요)

```
"리페인팅 방지(🔒)"가 켜져 있을 때:

  신호는 봉이 마감된 후에 나타남
  → 진입은 그 다음 봉의 시작가(시가)에!
```

### 구체적 예시

```
[5분봉 차트, 비트코인]

10:00~10:05 봉이 마감됨
  → 이 봉에 "▲ 매수" 신호가 확정됨!

10:05 새 봉 시작
  → 이 순간의 가격(시가)에 매수 진입!

이것이 "다음 봉 시가 진입"입니다.
```

### 타임프레임별 실제 대기 시간

| 분봉 | 신호 확정 시점 | 진입 시점 | 대기 시간 |
|------|-------------|----------|----------|
| 1분봉 | 매 1분마다 | 다음 1분 시작 | ~1분 |
| 5분봉 | 매 5분마다 | 다음 5분 시작 | ~5분 |
| 15분봉 | 매 15분마다 | 다음 15분 시작 | ~15분 |
| 1시간봉 | 매 정각 | 다음 정각 | ~1시간 |
| 4시간봉 | 4시간마다 | 다음 4시간 시작 | ~4시간 |
| 일봉 | 매일 09:00(한국) | 다음 날 09:00 | ~24시간 |

### 대시보드의 "다음행동" 항목

대시보드 맨 아래에 "다음행동" 줄이 있습니다:
```
⚡다음봉 시가에 진입!  ← 신호가 방금 나왔을 때
⏳신호 대기중         ← 아직 신호 없음
📊보유 유지           ← 포지션 있고, 신호 없음
⚡다음봉 시가에 청산!  ← 청산 신호 나왔을 때
```

## 5. ★ 신호 완벽 이해

### 모든 신호 종류

| 신호 | 모양 | 색상 | 의미 | 행동 |
|------|------|------|------|------|
| ▲ 매수 | 위쪽 삼각형 | 초록 | 상승추세 시작 | **다음봉 시가에 롱 진입** |
| ▲+ 강매수 | 큰 위쪽 삼각형 | 진한 초록 | 강한 상승 (볼륨 확인) | **더 확신 있게 롱 진입** |
| ▼ 매도 | 아래쪽 삼각형 | 빨강 | 하락추세 시작 | **롱 청산 (선물: 숏 전환)** |
| ▼+ 강매도 | 큰 아래쪽 삼각형 | 진한 빨강 | 강한 하락 | **즉시 청산/숏 진입** |
| ✕ 청산 | 작은 X | 주황/파랑 | 이익실현 타이밍 | **보유물량 매도** |
| ◆ 반전매수 | 다이아몬드 위 | 파랑 | 바닥 가능성 | 역추세 매수 |
| ◆+ 강반전 | 큰 다이아몬드 | 진한 파랑 | 강한 바닥 | 역추세 강매수 |
| ◆ 반전매도 | 다이아몬드 아래 | 주황 | 천장 가능성 | 역추세 매도 |

### 현물 모드에서의 차이

| 신호 | 선물에서 | 현물(업비트 등)에서 |
|------|---------|------------------|
| ▲ | 롱 진입 | 매수 |
| ▼ | 숏 진입 | **보유물량 전액 매도** |
| ✕ | 포지션 청산 | 보유물량 매도 |

현물에서 ▼ 신호 = "숏을 치라"는 게 아니라 "가지고 있는 것을 팔아라"입니다.

## 6. ★ 강매수/강매도 vs 일반 — 숫자의 의미

### 강도 숫자 (1~4)

화살표 안에 있는 숫자는 "이 신호가 얼마나 믿을 만한가"입니다.

```
1 = 약한 신호 → 반전/되돌림 가능성 있음 → 소량만 진입
2 = 보통 신호 → 기본 진입
3 = 강한 신호 → 추세 지속 가능성 높음 → 확신 있게 진입
4 = 매우 강한 신호 → 추세+볼륨+모멘텀 모두 확인 → 풀 진입
```

### 숫자별 추천 행동

| 강도 | 진입 비율 | SL 설정 |
|------|----------|---------|
| 1 | 전체 자금의 20~30% | 좁게 (빠른 손절) |
| 2 | 전체 자금의 40~50% | 기본 |
| 3 | 전체 자금의 60~70% | 기본 |
| 4 | 전체 자금의 70~80% | 넓게 (추세 신뢰) |

### "+" 표시 (강매수/강매도)

"+"가 붙으면 볼륨과 모멘텀이 동시에 확인된 것입니다.
```
▲ (일반) = MACD 크로스만 감지
▲+ (강한) = MACD 크로스 + RSI 확인 + 볼륨 확인 + 모멘텀 확인
```

**무조건 숫자 높을 때만 들어가야 하나요?**
→ 아닙니다. 하지만 초보자라면 **강도 3~4이고 +가 붙은 신호만** 따라가는 것을 추천합니다.

## 7. ★ 반대 신호가 나올 때

### 시나리오: ▲ 매수 후 ▼ 매도가 나옴

```
방법 A: 즉시 반대 전환 (선물, 공격적)
  ① 롱 전량 청산
  ② 바로 숏 진입
  장점: 큰 추세 전환을 놓치지 않음
  단점: 횡보장에서 왔다갔다 → 수수료 손실

방법 B: 청산만, 대기 (초보자 강력 추천!)
  ① 롱 전량 청산
  ② 새로운 확실한 신호 나올 때까지 관망
  장점: 횡보장 손실 최소화
  단점: 큰 하락 추세를 못 탈 수 있음

방법 C: Smart Trail 기준 (중급자 추천)
  ① 가격이 Smart Trail 아래로 갔으면 → 청산
  ② Smart Trail 위에 있으면 → 유지 (가짜 신호 가능)
```

### 시나리오: 아무 신호 없이 계속 손실

```
① SL 라인이 있으면 → SL 도달 시 무조건 청산!
② Smart Trail이 반전하면 → 청산
③ 둘 다 없으면 → 미리 정한 %에서 손절 (예: -2%)

⚠️ "조금만 더 기다리면..."은 가장 위험한 생각입니다!
```

## 8. ★ ✕ 청산 신호

### 전부 파나요?

```
초보자: 네! 전량 청산하세요. (가장 안전)
중급자: 50~70% 청산, 나머지는 TP2까지 보유
고급자: 무시하고 Smart Trail이 꺾일 때까지 보유 가능
```

### "포지션 없는데 ✕가 뜨는" 문제 → v4에서 해결!

v3까지는 포지션 유무와 관계없이 기술적 조건만 보고 ✕를 표시했습니다.
**v4에서는 포지션이 있을 때만 ✕가 표시됩니다.**

```
v3(이전): ▲ 없이도 ✕ 나옴 → 혼란
v4(현재): ▲ 진입 → ✕ 나옴 → 청산 → 포지션 없음 → ✕ 안 나옴 ✅
```

## 9. 리페인팅 (신호가 사라지는 문제)

**원인:** 봉이 마감되기 전에 신호를 계산하면, 가격 변동에 따라 신호가 바뀔 수 있음

**해결:** "🔒 리페인팅 방지"를 켜면 봉 마감 후에만 신호가 나타나서 절대 사라지지 않음

**확인법:** 상단 포지션바에 "🔒확정봉" 표시 확인

## 11. [2] 기본 설정

### 🎚️ 민감도 (5~30) — 가장 중요!

이 숫자는 신호를 만드는 MACD의 기간을 결정합니다.

```
숫자가 낮으면 (5~10):
  → 짧은 이동평균을 사용 → 가격 변화에 빠르게 반응
  → 신호가 자주 나옴 → 가짜 신호도 많음
  → 스캘핑/초단타에 적합

숫자가 높으면 (20~30):
  → 긴 이동평균을 사용 → 큰 추세만 감지
  → 신호가 드물게 나옴 → 신뢰도 높음
  → 스윙/장기에 적합
```

**비유:** 그물의 눈 크기
- 민감도 8 = 촘촘한 그물 → 작은 물고기(작은 추세)도 잡음 → 쓰레기도 많이 걸림
- 민감도 22 = 큰 그물 → 큰 물고기(큰 추세)만 잡음 → 놓치는 것도 있음

### ⚡ 민첩성 (1~10)

민감도와 별개로, 신호 발동 조건의 엄격함을 조절합니다.

```
1~3 (느림): MACD가 확실히 크로스해야만 신호 → 확실한 진입점
4~6 (중간): 기본
7~10 (빠름): MACD 크로스 외에 모멘텀 변화로도 신호 발동 → 빠르지만 노이즈↑
```

**비유:** 출발 총소리
- 민첩성 2 = "준비... 땅!" 확실히 들린 후 출발
- 민첩성 8 = 발소리만 들려도 출발

## 12. [3] 필터 설정 — 필터별 매매 기법

### 필터란?
"신호를 걸러내는 체"입니다. 체가 없으면 모든 신호가 다 나오고,
체를 추가할수록 통과하는 신호만 남습니다.

### 필터별 매매 기법

**🛡️ Smart Trail 필터 + Confirmation 모드**
```
효과: 상승추세일 때만 매수, 하락추세일 때만 매도
매매법:
  1. Smart Trail이 파란색 → ▲ 대기
  2. ▲ 발생 → 롱 진입 (Smart Trail = 손절선)
  3. Smart Trail 빨간 전환 → 전량 청산
최적: 중기 추세추종 (1시간~일봉)
```

**💪 추세 강도 필터 단독**
```
효과: ADX>25인 추세장에서만 신호
매매법:
  1. 대시보드에 "추세강도: 보통/강함" 표시될 때만 매매
  2. "횡보" 표시되면 매매 중단
  3. 신호가 나오면 기본대로 진입
최적: 횡보장 손실 회피
```

**📐 Trend Tracer 필터 + Smart Trail**
```
효과: 이중 확인 — 큰 추세 + 세부 추세 모두 같은 방향
매매법:
  1. Trend Tracer 파란색 + Smart Trail 파란색 = 상승 확정
  2. ▲ 발생 → 확신 있는 롱 진입
  3. 둘 중 하나라도 빨간 전환 → 일부 청산
최적: 안전한 스윙 트레이딩
```

**☁️ Neo Cloud 필터**
```
효과: 구름 위에서만 매수, 아래에서만 매도
매매법:
  1. 가격이 구름 위에 있을 때 → 매수 신호만 따름
  2. 구름 안에 있을 때 → 매매 보류
  3. 구름 아래로 이탈 → 전량 청산
최적: 큰 추세 필터링 (4시간~일봉)
```

## 13. [4] 캔들 색상 — 완전 이해

### 왜 헷갈리나요?

```
트레이딩뷰 기본: 상승봉=초록, 하락봉=빨강
한국 주식(키움/업비트): 상승봉=빨강, 하락봉=파랑

이 지표의 캔들 색상은 위의 것과 다릅니다!
이 색상은 "이 봉이 올랐나 내렸나"가 아니라
"현재 전체적인 추세 상태"를 보여줍니다.
```

### 추세 3색 (기본, 추천)

```
🟢 밝은 초록 = 강한 상승 추세 중 (MACD 양수 + 히스토그램 양수)
🟢 연한 초록 = 약한 상승 추세 중 (MACD 양수이지만 약해지는 중)
🟣 보라     = 추세 전환 중 또는 방향 불명
🔴 연한 빨강 = 약한 하락 추세 중
🔴 밝은 빨강 = 강한 하락 추세 중

※ 초록 봉이라고 해서 "이 봉이 올랐다"는 뜻이 아닙니다!
   "전체 추세가 상승 중"이라는 뜻입니다.
   하락한 봉도 추세가 상승이면 초록으로 칠해집니다.
```

### 읽는 법

```
▲ 매수 후 캔들이 계속 초록 → 추세 건강함 → 보유 유지 ✅
▲ 매수 후 캔들이 보라로 변함 → 추세 약해짐 → 주의! ⚠️
▲ 매수 후 캔들이 빨강으로 변함 → 추세 꺾임 → 청산 고려 🔴
```

### "끄기" 옵션

색상이 혼란스러우면 "끄기 (원래 색상)"을 선택하세요.
트레이딩뷰의 기본 캔들 색상(초록/빨강 또는 빨강/파랑)이 그대로 나옵니다.

## 14. [5] 오버레이 지표

### Smart Trail (추적 손절선) — 가장 중요

```
무엇: 가격을 따라다니는 동적 손절선
파란선+파란영역 = 상승추세 (이 선 = 나의 손절 라인)
빨간선+빨간영역 = 하락추세 (이 선 = 저항선)

사용법:
  롱 진입 시 → Smart Trail 값을 손절가로 거래소에 입력
  선이 파→빨으로 바뀌면 → 무조건 청산

민감도 1~2: 가격에 가깝게 → 빠른 반응 (스캘핑)
민감도 3: 기본 (데이트레이딩)
민감도 4~5: 가격에서 멀게 → 느린 반응 (스윙)

ATR 배수 0.5~1.0: 좁은 손절 (빠른 손절, 작은 손실)
ATR 배수 1.0~2.0: 넓은 손절 (큰 흔들림 견딤, 큰 손실 가능)
```

### Reversal Zones (반전 영역)

```
빨간 영역(위): 과매수 → 이익실현 또는 매도 고려
파란 영역(아래): 과매도 → 매수 고려
회색 점: 중심선

주의: 강한 추세에서는 영역을 뚫고 갈 수 있음!
가격이 영역 안에서 "되돌아올 때" 행동하세요.
```

### Trend Tracer vs Trend Catcher

```
Trend Tracer: 큰 강물의 방향 (안정적, 느림)
Trend Catcher: 물살이 바뀌는 순간 (민감, 빠름)

둘 다 같은 방향 → 강한 추세 확인!
Catcher만 전환 → 아직 확실하지 않음
Catcher 전환 → Tracer도 전환 → 새 추세 확정!
```

### Neo Cloud

```
초록 구름: 상승추세 (구름 = 지지)
빨간 구름: 하락추세 (구름 = 저항)
밝은색: 새로운 추세
흐린색: 오래된 추세

가격이 구름 위 → 매수 위주
가격이 구름 안 → 관망
가격이 구름 아래 → 매도 위주
```

## 15. [6] 지지/저항선

### 무엇인가?
가격이 반복적으로 튕겨나가는 수평선입니다.

```
S (파란 점선): 지지선 — 가격이 여기서 반등할 가능성
R (빨간 점선): 저항선 — 가격이 여기서 막힐 가능성
```

### 사용법 (넥라인 활용)

```
방법 1: 지지선 반등 매수
  가격이 S 라인 근처 + ▲ 신호 → 강한 매수 근거
  SL은 S 라인 아래로 설정

방법 2: 저항선 이탈 매수 (돌파 매매)
  가격이 R 라인을 위로 돌파 + ▲ 신호 → 매수
  이전 R 라인이 새로운 S 라인이 됨

방법 3: 저항선 반락 매도
  가격이 R 라인에서 막힘 + ▼ 신호 → 매도/숏
```

### 피봇 길이 설정

```
짧은 분봉(1~15분): 피봇 길이 5~10
긴 분봉(1시간~일봉): 피봇 길이 15~30
표시 개수: 2~3개 추천 (너무 많으면 복잡)
```

## 16. [7] TP/SL

### 레버리지 자동 조절 기능

```
레버리지 배율에 따라 TP/SL이 자동으로 좁아집니다.

예: ATR 기반, TP1 배수 1.5, SL 배수 1.0

1배(현물):  TP1 = ATR × 1.5 (100%)
3배:       TP1 = ATR × 1.5 × 0.58 (58%)
5배:       TP1 = ATR × 1.5 × 0.45 (45%)
10배:      TP1 = ATR × 1.5 × 0.32 (32%)
20배:      TP1 = ATR × 1.5 × 0.22 (22%)
50배:      TP1 = ATR × 1.5 × 0.14 (14%)
100배:     TP1 = ATR × 1.5 × 0.10 (10%)

공식: 원래값 × (1 / √레버리지)
```

## 24. 자주 묻는 질문

**Q: 색이 겹쳐서 안 보여요 (오버레이가 너무 많으면)**
A: 오버레이는 2~3개만 동시에 켜세요.
   권장 조합: Smart Trail + (Trend Tracer 또는 Reversal Zones)
   Neo Cloud + Reversal Zones을 동시에 켜면 겹칩니다 → 하나만 선택

**Q: SL(로스컷) 라인이 안 보여요**
A: TP/SL은 매수/매도 신호가 발생한 후에만 나타납니다.
   신호 없이는 TP/SL 라인도 없습니다.
   또한 "모습(Style)" 탭에서 SL 플롯이 보이기로 되어 있는지 확인하세요.

**Q: 현물에서 매도 신호가 없게 하고 싶어요**
A: 시장 유형을 "코인 현물" 또는 "한국주식"으로 설정하면 
   ▼가 "숏 진입"이 아닌 "매도(청산)"으로 바뀝니다.
   완전히 숨길 수는 없지만, 현물에서의 ▼ = "가진 거 파세요"입니다.

**Q: 지지/저항선(넥라인)으로 진입이 가능한가요?**
A: 네! [6] 지지/저항선을 켜면 S/R 라인이 표시됩니다.
   ▲ 신호 + S 라인 근처 = 매우 강한 매수 근거
   자세한 방법은 Part 3의 15번 참고

**Q: 금/은/유가에도 되나요?**
A: 네! 시장 유형에서 "금/은/원자재" 선택하세요.
   추세가 길어서 높은 민감도(18~25)를 추천합니다.

**Q: 이 지표만으로 돈을 벌 수 있나요?**
A: 어떤 지표도 단독으로 수익을 보장하지 않습니다.
   리스크 관리(손절!), 자금 관리, 심리 관리가 지표보다 중요합니다.

## 25. 색 겹침 / SL 안 보임 해결

### 오버레이 겹침 해결

```
문제: Neo Cloud + Reversal Zones + Smart Trail 다 키면 색이 겹침
해결: 동시에 2개까지만 켜세요

추천 조합:
  초보자: Smart Trail만
  중급자: Smart Trail + Trend Tracer
  고급자: Smart Trail + Reversal Zones
  역추세: Reversal Zones + Neo Cloud

※ "모습(Style)" 탭에서 각 오버레이의 투명도를 조절할 수도 있습니다
```

### TP/SL 안 보일 때

```
원인 1: 아직 매수/매도 신호가 없음 → 신호 발생 후 나타남
원인 2: TP2 또는 SL에 이미 도달해서 초기화됨
원인 3: Style 탭에서 숨겨져 있음 → TP1, TP2, SL 체크
원인 4: 현물 모드에서 ▼(매도) 후에는 TP/SL 초기화됨
```
